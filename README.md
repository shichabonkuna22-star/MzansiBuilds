# 🚀 MzansiBuilds - Build in Public Platform

**Derivco Code Skills Challenge 2026** | Stage 2 to Stage 3 Submission

## 📋 Project Overview

MzansiBuilds is a platform that helps developers build publicly and keep up with what other developers are building. Developers can create projects, track milestones, collaborate, and get celebrated upon completion.

## 🎯 Competencies Demonstrated

| Competency | Implementation |
|------------|----------------|
| **Project Profiling** | Each project has title, description, stage, support_needed, timestamps |
| **Code Version Control** | Semantic commits (feat:, docs:, test:, security:) |
| **Test-Driven Development** | pytest tests in `/tests` directory |
| **Secure By Design** | Password hashing (bcrypt), CSRF tokens, SQL injection prevention (ORM) |
| **Documentation** | This README + inline comments + docstrings |
| **Ethical Use of AI** | Comments marked `# AI_ASSIST:` and `# HUMAN_DECISION:` |

## 🛠 Tech Stack

- **Backend**: Python Flask (MVC pattern - C#/ASP.NET equivalent)
- **Database**: SQLite (production-ready for PostgreSQL with MZB_ prefix)
- **Frontend**: Bootstrap 5 + Custom CSS (Green/White/Black theme)
- **Authentication**: Flask-Login with bcrypt
- **Testing**: pytest

## 🎨 Design Theme

- **Green**: `#2E7D32` (Primary actions, success states)
- **White**: `#FFFFFF` (Backgrounds, cards)
- **Black**: `#212121` (Text, footers)

## ✨ Features

1. **User Accounts** - Register/Login with profile pictures
2. **Project Management** - Create, view, update, delete projects
3. **Progress Tracking** - Add milestones to track achievements
4. **Live Feed** - See what others are building with pagination
5. **Collaboration** - Comment on projects + "Raise Hand" for collab
6. **Celebration Wall** - Completed projects with confetti animation
7. **Animations** - Slide-up, slide-right, hover effects
